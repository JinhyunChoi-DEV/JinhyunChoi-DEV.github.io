## Description
- ##### CamPic is a 2D platformer puzzle game that player uses their camera to capture themselves and use them as a alternative platform to proceed to the goal.

<br>
<br>

## Information
- ###### **Platform**: [PC](https://drive.google.com/file/d/1Fltqv4KDaZuk__XrF2JNeBuhomWoj_te/view?usp=sharing)
- ###### **Engine**: Custom Engine
- ###### **Team Size**: 4 persons
- ###### **Genre**: 2D Platformer Puzzle Game
- ###### **Language**: C++
- ###### **Time Spent On Project**: 1 Year (Sept 2018 ~ June 2019)
- ###### **My Role**: Tech Director, Engine Architecture, Graphics Programmer, Main Programmer

<br>
<br>

## Summary Of Contributions
 - ###### As a Tech Director, I should have controlled all of our team project codes. For managing codes, I studied git and source control systems. This experience made me understand and used effectively git and source control.
 - ###### As an Engine Architecture, I should have made a game engine system. For making a game engine, I studied engine architecture and engine flow systems with many game engine explanation books and articles. After studying game engine systems. I made my 2D custom engine.
 - ###### As a Graphics Programmer, I should have made our engine's graphics system. I studied OpenGL for applying our engine. After studing, I made Sprite, Animation, Particle, ScreenShot, and many other 2D grahpics system.
 - ###### As a Main Programmer, I should have made our game's main feature, a camera system. For making this system, I studied 2D game physics. After studing, I made player capture, zoom-in, zoom-out, and other main feature in our game.
