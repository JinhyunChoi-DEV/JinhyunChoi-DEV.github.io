## Description
- ##### A 2D puzzle-platform game where character’s color changes physics of the world.<br/>Players must cleverly change character’s colors in order to escape the level.

<br>
<br>

## Information
- ###### **Platform**: [PC](https://drive.google.com/file/d/199CgRWj6YCOg3JYwQ0xes0QpoleFjm6B/view?usp=sharing)
- ###### **Engine**: Givien Engine (For Students In Our College)
- ###### **Team Size**: 4 persons
- ###### **Genre**: 2D Platformer Puzzle Game
- ###### **Language**: C++
- ###### **Time Spent On Project**: 3 Month (Mar 2018 ~ June 2018)
- ###### **My Role**: Test Manager, Game Function Creator, Sub Tech Programmer

<br>
<br>

## Summary Of Contributions
 - ###### As a Sub Tech Programmer, I should have made our game's main system, a rope action system. For making a rope action system, I studied our engine's physics code and customize it for our rope action system.
 - ###### As a Test Manager, I should have checked if our game had bugs or error. Moreover, If I found that, I should have fixed it as a Test manager role. During this project, I studied debugging tips and got experience in effective debug.
 - ###### As a Game Function Creator, I should have made subsystems. Therefore, I made UI and Outgame Stages. For making these systems, I studied our engine's flow system like Stage Manager, Engine System code, and other things.
